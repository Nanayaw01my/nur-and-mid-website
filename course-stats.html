1. Descriptive Statistics summarizing data

1.1 Measures of Central Tendency

  • Mean (μ or x̄): arithmetic average. Sensitive to outliers. Formula: Σx / n.
  • Median: middle value when data ordered. Robust to outliers. Use for skewed distributions (e.g., length of stay).
  • Mode: most frequent value. Useful for nominal data (e.g., blood type).

Nursing example: In a study of pain scores (0‑10) post‑op: mean 4.2, median 3.5, mode 2 (bimodal distribution).

1.2 Measures of Dispersion (Variability)

  • Range: maximum – minimum. Simple but unstable.
  • Interquartile range (IQR): Q3 – Q1 (middle 50%). Used with median for skewed data.
  • Variance (σ² or s²): average squared deviation from mean. Formula: Σ(xᵢ – x̄)² / (n‑1) for sample.
  • Standard deviation (SD, s): square root of variance. Same units as original data. For normally distributed data, ≈68% within 1 SD, 95% within 2 SD.

1.3 Frequency Distributions & Visualizations

  • Histogram: continuous data, bars touch. Assess shape: normal, skewed (left/right), bimodal, uniform.
  • Boxplot (box & whisker): shows median, IQR, range, outliers (points >1.5×IQR).
  • Bar chart: categorical data (frequencies).
  • Pie chart: proportions – use sparingly (hard to compare).
  • Stem‑and‑leaf: retains individual values.

1.4 Shape of Distribution

  • Skewness: left (negative) – mean < median; right (positive) – mean > median.
  • Kurtosis: peakedness – leptokurtic (heavy tails), platykurtic (light tails), mesokurtic (normal).
  • Normality tests: Shapiro‑Wilk, Kolmogorov‑Smirnov (p >0.05 suggests normal).

2. Inferential Statistics drawing conclusions

2.1 Hypothesis Testing Framework

  • Null hypothesis (H₀): no effect/difference (e.g., mean BP equal between groups).
  • Alternative hypothesis (H₁ or Hₐ): there is an effect (two‑tailed: different; one‑tailed: greater/less).
  • Type I error (α): rejecting true H₀ (false positive). Usually set α = 0.05.
  • Type II error (β): failing to reject false H₀ (false negative). Power = 1 – β (want ≥0.80).
  • p‑value: probability of obtaining observed results (or more extreme) if H₀ is true. If p < α, reject H₀.

2.2 Confidence Intervals (CI)

Range of plausible values for population parameter. 95% CI means: if we repeated study 100 times, 95 intervals would contain true value.

Interpretation: CI that does not cross null value (e.g., 0 for difference, 1 for ratio) indicates statistical significance.

2.3 Parametric Tests

Assume normality, homogeneity of variance, independence.

  • One‑sample t‑test: compares sample mean to known population mean. Formula: t = (x̄ – μ) / (s/√n).
  • Independent samples t‑test: compares means of two independent groups. Assumes equal variances (Levene's test). If unequal, use Welch's t‑test.
  • Paired t‑test: compares means of same group at two times (e.g., pre‑test/post‑test).
  • One‑way ANOVA: compares means of ≥3 groups. F = between‑group variance / within‑group variance. Post‑hoc tests (Tukey, Bonferroni) if significant.
  • Repeated measures ANOVA: same subjects measured multiple times.
  • Two‑way ANOVA: two independent variables (factors) – tests main effects and interaction.

2.4 Non‑parametric Tests (distribution‑free)

Used when assumptions violated, ordinal data, or small samples.

  • Mann‑Whitney U: independent groups (alternative to independent t‑test).
  • Wilcoxon signed‑rank: paired data (alternative to paired t‑test).
  • Kruskal‑Wallis H: one‑way ANOVA alternative.
  • Friedman test: repeated measures ANOVA alternative.
  • Chi‑square (χ²): tests association between categorical variables. Also goodness‑of‑fit. Expected frequency ≥5 per cell. Yates' continuity correction for 2×2 tables.
  • Fisher's exact test: for 2×2 when expected <5.

3. SPSS Basics data analysis in nursing research

3.1 Data Entry & Variable View

  • Variable View: define variables: Name (no spaces), Type (numeric, string), Label (description), Values (coding e.g., 1=Male, 2=Female), Measure (scale, ordinal, nominal).
  • Data View: enter data in rows (cases) and columns (variables).
  • Missing values: define as 999 or leave blank; system missing is period.

3.2 Running Descriptives

  • Analyze → Descriptive Statistics → Frequencies: for categorical variables (frequencies, bar charts).
  • Analyze → Descriptive Statistics → Descriptives: for continuous (mean, SD, min, max, skewness, kurtosis).
  • Analyze → Descriptive Statistics → Explore: detailed stats, boxplots, normality tests (Shapiro‑Wilk).
  • Analyze → Compare Means → Means: group means and ANOVA table.

3.3 Interpreting SPSS Output

  • p‑value (Sig.): if <0.05, statistically significant.
  • Levene's test: if p <0.05, equal variances not assumed (use Welch t‑test row).
  • ANOVA table: F statistic, df, Sig. Post‑hoc: multiple comparisons.
  • Chi‑square: Pearson Chi‑Square value, df, Asymptotic Sig. (2‑sided).
  • Correlation: Pearson's r (strength: 0‑0.3 weak, 0.3‑0.7 moderate, >0.7 strong), Sig. (2‑tailed).

3.4 Creating Graphs in SPSS

  • Graphs → Chart Builder: drag variables, choose chart type (histogram, boxplot, bar, scatter).
  • Graphs → Legacy Dialogs: simpler interface.
  • Editing: double‑click chart to open Chart Editor – add titles, change colors, adjust scales.

4. Correlation & Regression relationships & prediction

4.1 Pearson Product‑Moment Correlation (r)

Measures linear relationship between two continuous variables. Range ‑1 to +1. Assumptions: linearity, bivariate normality, no outliers.

Interpretation: r = 0.8 strong positive; r = ‑0.5 moderate negative; r = 0 no linear relationship.

Coefficient of determination (R²): proportion of variance shared. r=0.5 → R²=0.25 → 25% variance explained.

4.2 Spearman's Rank Correlation (ρ)

Non‑parametric alternative. Uses ranks. For ordinal data or when assumptions violated.

4.3 Simple Linear Regression

Predicts continuous outcome (Y) from one predictor (X). Equation: Y = b₀ + b₁X + ε.

  • b₀ (intercept): predicted Y when X=0.
  • b₁ (slope): change in Y per 1‑unit increase in X.
  • R²: % variance in Y explained by X.
  • ANOVA for regression: tests if model significantly predicts Y (p <0.05).
  • Residuals: should be normally distributed, homoscedastic (constant variance).

4.4 Multiple Linear Regression

Several predictors (X₁, X₂…). Adjusted R² penalizes for many predictors. Check multicollinearity (Variance Inflation Factor, VIF <10).

Nursing example: predicting length of hospital stay from age, comorbidities, admission acuity, social support.

4.5 Logistic Regression

For binary outcome (e.g., readmitted yes/no). Reports odds ratios (OR) and 95% CI. Goodness‑of‑fit: Hosmer‑Lemeshow test.

5. Effect Sizes practical significance

5.1 Why Effect Sizes?

p‑values depend on sample size. Effect size quantifies magnitude of effect, independent of n. Essential for meta‑analysis and power analysis.

5.2 Cohen's d (for t‑tests)

d = (mean₁ – mean₂) / pooled SD. Interpretation (Cohen): small 0.2, medium 0.5, large 0.8.

Formula pooled SD: √[(s₁² + s₂²) / 2].

5.3 Eta‑squared (η²) for ANOVA

η² = SS_between / SS_total. Proportion of variance explained. Small 0.01, medium 0.06, large 0.14.

Partial η²: for factorial ANOVA (effect size adjusted for other factors).

5.4 Cohen's w / Cramér's V for Chi‑square

Cramér's V: for contingency tables (0‑1). Small 0.1, medium 0.3, large 0.5.

5.5 Odds Ratio (OR) & Relative Risk (RR)

OR: odds of outcome in exposed / odds in unexposed. OR=1 no effect; >1 increased risk; <1 protective. Used in case‑control studies.

RR: risk in exposed / risk in unexposed. Used in cohort studies.

5.6 Number Needed to Treat (NNT)

NNT = 1 / absolute risk reduction (ARR). Number of patients needed to treat to prevent one adverse outcome. Ideal NNT low (e.g., 2‑10).

6. Presenting Data – APA 7th Edition tables & figures

6.1 General APA Guidelines

  • Double‑space, serif font (Times New Roman 12).
  • Margins 1 inch.
  • Page numbers top right.

6.2 Tables

  • Table number (bold) above table: Table 1
  • Title (italic, title case): Demographic Characteristics of Study Participants
  • Horizontal lines only (no vertical lines).
  • Clear column headings, units specified.
  • Notes below for abbreviations, significance levels: *p < .05. **p < .01.

Example table:

6.3 Figures

  • Figure number below figure: Figure 1. Mean Pain Scores Over Time by Group
  • Axes clearly labeled, legend if needed.
  • Use high contrast, avoid 3D effects.
  • Error bars = SD or 95% CI (specify in caption).

6.4 Reporting Statistics in Text

  • t(38) = 2.24, p = .031, d = 0.71
  • F(2, 87) = 5.67, p = .005, η² = 0.12
  • χ²(1, N=120) = 4.83, p = .028, V = 0.20
  • r(45) = .42, p = .003, 95% CI [.16, .63]
  • OR = 2.5, 95% CI [1.3, 4.8]

6.5 Common Mistakes

  • Confusing statistical significance with clinical significance (always report effect size).
  • Not checking assumptions before running tests.
  • Over‑interpreting non‑significant results (absence of evidence ≠ evidence of absence).
  • Using parametric tests for small, non‑normal samples.

7. Power Analysis & Sample Size Calculation planning studies

7.1 Key Concepts

Power (1‑β): probability of detecting an effect if it truly exists. Usually set at 0.80.

Effect size: expected magnitude (based on pilot, prior research, or clinical judgment).

α (significance level): usually 0.05.

Sample size: larger n increases power.

7.2 Software for Power Analysis

  • G*Power (free): t‑tests, ANOVA, correlation, regression, χ².
  • PASS, nQuery: commercial.
  • SPSS: no built‑in power, but can compute post‑hoc.

7.3 Example using G*Power

Independent t‑test, two‑tailed, α=0.05, power=0.80, medium effect (d=0.5) → required n=64 per group (total 128).
